# GSE164073 RNA-seq Analysis

Analysis of RNA-seq dataset [GSE164073](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE164073) (GRCh38.p13) including expression exploration, differential analysis, and visualization.

## Figures
  
- `plots/` – generated figures:
  - Dot plot  
  - Top 5 gene expression  
  - Top 20 gene heatmap  
  - MA plot  
  - Module-trait relationship  
  - PCA plot  
  - Soft threshold  
  - Top 10 gene total counts  
  - Volcano plot  
- `scripts/` – analysis R script  

## Requirements

- R ≥ 4.2  
- Packages: `DESeq2`, `WGCNA`, `ggplot2`, `pheatmap`, `EnhancedVolcano`, `dplyr`
